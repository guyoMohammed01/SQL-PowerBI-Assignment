# SQL-PowerBI-Assignment

## Project Overview
This project demonstrates SQL querying and Power BI data visualization skills using a sales and inventory database.

## Files Included
- `assignment_complete.sql` - Complete SQL queries solving 100 questions
- `Sales_Inventory_Dashboard.pbix` - Interactive Power BI dashboard

## Dashboard Features
The Power BI dashboard analyzes:
- **Sales Performance** - Total sales and trends over time
- **Product Performance** - Category analysis and top products
- **Customer Insights** - Sales by membership tier
- **Inventory Insights** - Stock alerts and reorder recommendations

## Database Schema
The database contains 4 tables:
- **customers** - Customer information and membership status
- **products** - Product catalog with pricing and categories
- **sales** - Transaction records
- **inventory** - Current stock levels

## Built With
- PostgreSQL (Database)
- SQL (Data querying)
- Power BI Desktop (Visualization)
